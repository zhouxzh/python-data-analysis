# Pandas 初学者教程项目

完整的 Pandas 学习资源，包含教程、示例代码、练习题和测试。

## 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行示例
python examples/01_basic_read.py

# 3. 完成练习
python exercises/exercise_01.py

# 4. 查看答案
python exercises/solution_01.py

# 5. 运行测试
pytest tests/
```

## 内容

- **教程文档**: [pandas.md](pandas.md)
- **测试数据**: data/ 目录
- **示例代码**: examples/ 目录（4个示例）
- **练习题**: exercises/ 目录（2个练习+答案）
- **测试代码**: tests/ 目录

## 学习顺序

1. 阅读 [pandas.md](pandas.md) 教程
2. 运行 examples/ 中的示例代码
3. 完成 exercises/ 中的练习题
4. 运行测试验证理解
